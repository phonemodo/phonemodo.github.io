How to Download Jojoy:

One reason why Jojoy is so popular is that it works on iOS Android, and desktop computers too. Installing it is simple – pick your platform and follow the steps:

Method 1: iOS

Download Link

  1. Open Safari browser and navigate to jojoy.org/ios/
  2. Download one of the configuration files and tap on Allow on the confirmation message
  3. When the Install Profile window opens, tap Install
  4. Input your device passcode if requested and tap Install
  5. Look on your home page – the Jojoy icon is there when the installation is finished.
  6. Open iOS Settings and tap General > Profiles
  7. Tap the Jojoy profile and tap Trust
  8. Close Settings – Jojoy is ready to use

Method 2: Android

Download Link

  1. Go into your Android settings app
  2. Tap on Privacy or Security
  3. Find and enable the Unknown Sources option
  4. Close Settings
  5. Open your browser and go to jojoy.org/apk/
  6. Download the APK file
  7. Open your device downloads folder and double-tap the file
  8. Follow the on-screen instructions to install and you will see the icon on your home screen

Method 3: PC or Mac

Because Jojoy is a mobile app installer, it doesn’t support desktop operating systems. However, with an Android emulator, such as Nox Player or BlueStacks, installed on your Windows or macOS computer, you can install Jojoy:

  1. Download an Android emulator onto your computer – we recommend Nox Player or BlueStacks
  2. Open the emulator and sign in – you will need a free Google account for this
  3. Download the Android APK file onto your computer from jojoy.org/apk/
  4. Once the emulator has finished setting up, open the search bar and type Jojoy – click on the result to get it into the emulator
  5. Alternatively, find the .apk file and drag it into the emulator or right-click it and click Open With
  6. Click the file in the emulator and wait for it to be installed
  7. Once done you can access Jojoy via the emulator

How to Use Jojoy:

Jojoy is easy to use:

  1. Launch Jojoy
  2. Use the search bar or browse to find an app or game to download
  3. Tap it and tap Install
  4. Follow the on-screen directions to install it and when you see the icon, your app is ready to use
